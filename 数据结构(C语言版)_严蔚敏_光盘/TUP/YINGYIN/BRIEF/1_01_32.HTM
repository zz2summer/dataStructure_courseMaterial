An Introduction to Object-Oriented Programming with
Java 2nd Edition
(Java面向对象程序设计 第2版）
作者：C.Thomas Wu
书号：ISBN 7-302-04220-9
开本：16开
定价：72.00元
内容简介：本书共分17章：第0章介绍计算机和程序设计语言：第1章引入了面向对象程序设计的基本概念；第2章Java程序设计初步；第3章介绍数值数据的操纵；第4章教你如何定义自己的类；第5章引入Applet(小应用程序)；第6章介绍if和Switch语句；第11章引入文件I/O；第12章介绍可重用类和包；第13章介绍GUI对象和事件驱动程序设计；第14章介绍继承和多态性以及如何使用他们进行高级程序设计；第15章为典型实例章，用本书所学到的技术开发大型软件；第16章介绍递归算法。附录是Javabook包。
本书比第1版增加并提供了所有典型实例程序和作者自定义的Javabook类的源代码，添加了Java 2平台的最新的链接列表、动态内存分配、Swing类、Vector类等以及galapagos软件包。全书用UML对象图来解释实例并可在Web站点下载具有动画的演示文稿及练习答案。
